# Spatial computing project

Project of the course *Foundations of Spatial Computing and Applications in Augmented and Virtual Reality* , Unversité de Fribourg, Spring semester 2026

### Introduction
The category of the project is _Dissemination & Education_ and is in the form of a desktop app.
The goal of the project is to have an interactive web page to present the functioning of a musical instrument present at [SMEM](https://www.smemmusic.ch/en) (Swiss museum of electronic music) to the public.
The proposed instrument is [Mellotron](https://it.wikipedia.org/wiki/Mellotron) M 400 

### Dependencies  
Javascript based web application. Node.js is the used environment and Vite handles the frontend build. The 3D object is uploaded thanks to the library three.js
